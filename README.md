Spam Email Classifier

This project implements a machine learning spam detection system using the
Apache SpamAssassin dataset.

Features

- Email parsing
- HTML cleaning
- Custom NLP preprocessing
- Stemming using NLTK
- Custom Scikit-learn transformers
- Logistic Regression classifier
- Pipeline based ML workflow

Dataset

Apache SpamAssassin Public Corpus

Tech Stack

Python
Scikit-learn
NLP
NLTK

Results

Precision: ~96%
Recall: ~94%
